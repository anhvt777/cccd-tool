# CCCD Tool

Trang tách riêng từ dự án Kehoachtuan để vận hành độc lập trên GitHub Pages.

## Sử dụng

- Mở link GitHub Pages của repo này.
- Dữ liệu QR/CCCD được xử lý trên trình duyệt, không gửi lên máy chủ.
- Nếu dùng camera trên iPhone/Android, cần mở bằng HTTPS.

## Bảo mật

Repo này không chứa API key, service role key, dữ liệu Supabase hoặc dữ liệu của web Kehoachtuan.
